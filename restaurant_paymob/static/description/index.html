Order Lifecycle

New → Confirmed → Preparing → Ready → Paid → Done

Every state change is logged automatically in the Chatter with timestamp and user name. Orders can be cancelled at any stage except Paid.

PayMob Integration Flow

1️⃣ Auth Token
POST /auth/tokens
→
2️⃣ Register Order
POST /ecommerce/orders
→
3️⃣ Payment Key
POST /payment_keys
→
✅ iFrame URL
Customer pays

After payment, PayMob sends a Webhook POST to /paymob/webhook. The module verifies the HMAC-SHA512 signature and automatically updates the order to Paid.

Quick Setup — 3 Steps

1️⃣

Install the Module

Copy the folder to your Odoo addons path, update the app list, and click Install.

2️⃣

Enter PayMob Credentials

Go to Settings → PayMob and enter your API Key, Integration ID, iFrame ID, and HMAC Secret.

3️⃣

Set Webhook URL

In PayMob Dashboard set the Transaction Callback URL to:
https://yourdomain.com/paymob/webhook
